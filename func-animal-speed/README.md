# 動物に乗って走ったら何時間？
東京から各都市まで動物に乗って走ったとしたら、何時間で走行できるかを
調べるプログラム

動物の最高時速と東京から各都市までの距離は以下の表にまとめる

## 動物の最高時速
|動物|最高時速|
|:----|:---------|
|チーター|110km/h|
|トナカイ|80km/h|
|シマウマ|60km/h|
|ライオン|58km/h|
|キリン|50km/h|
|ラクダ|30km/h|

## 東京から各都市までの距離
|都市|距離|
|:----|:---------|
|静岡|183.7km|
|名古屋|350.6km|
|大阪|507.5km|

## 公式
時速や距離から時間を求める公式
```text
時間 = 距離 ÷ 速度
```
